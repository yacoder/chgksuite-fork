# Word

![](images/word.png)

Откройте **4s-файл**, который у вас получился после [парсинга](../4s). После нажатия кнопки «Запустить» в той же папке, что и 4s-файл, появится файл в формате docx.

## Спойлеры

Полезны для случаев, когда надо отправить кому-то файл на тестирование. По умолчанию спойлеры отключены, они бывают трёх видов:

- `whiten` — спойлеры забелением. Неудобны при тестировании с мобильного.
- `pagebreak` — спойлеры переносом страницы. Гугл-докс вырезает переносы страницы на мобильном, поэтому могут быть тоже не идеальны.
- `dots` — спойлеры отбивкой точками. Выглядит не очень эстетично, зато работает, в том числе в гугл-докс с мобильного.

## Версия для экрана {: #screen }

Версия, в которой удалены ударения и указания ведущему в квадратных скобках, а также внесены изменения из [тега `screen`](../4s#screen). Нужна для онлайнов.

Существует в двух видах:

- `replace_all` — во всех вопросах вместо версии для ведущего выводит версию для экрана. При выборе этого варианта к имени файла будет добавлен фрагмент `_screen`.
- `add_versions` — каждый вопрос сначала приводится в версии для ведущего, а потом в версии для экрана. Удобно для случаев, когда не используются сервисы типа [OpenQuiz](https://open-quiz.com), а ведущий скидывает текст вопроса команде в чат. При выборе этого варианта к имени файла будет добавлен фрагмент `_screen_versions`.

- `add_versions_columns` — аналогично `add_versions`, но используется вёрстка в две колонки — слева версия для ведущего, справа для экрана.

Квадратные скобки можно эскейпить, если написать перед ними слеш, `\[вот так\]` — тогда они не будут удаляться.

## Дополнительные настройки

- **Шрифт** (`--font`) — заменить шрифт документа. Старый вариант `--font_face` тоже поддерживается.
- **Встроить шрифт** (`--embed_fonts on`) — положить выбранный через `--font` шрифт внутрь DOCX. Работает с установленным шрифтом или прямым путём к `.ttf`/`.otf`; коллекции `.ttc` нужно предварительно сохранить как отдельный файл шрифта.
- **Оптимизировать размер** (`--optimize_size`) — по умолчанию включено. Сжимает непрозрачные картинки в JPEG с качеством 80, если новый файл получается меньше; PNG с прозрачностью остаются PNG и сжимаются lossless с сохранением прозрачности. При `--embed_fonts on` встраивает только используемые в документе символы шрифта. Можно отключить: `--optimize_size off`.
- **Без ответов** (`--noanswers`) — не печатать ответы совсем.
- **Без переноса строки после «Вопрос N.»** (`--noparagraph`) — номер вопроса и текст вопроса на одной строке.
- **Без слова «Вопрос»** (`--only_question_number`) — вместо «Вопрос 1.» будет просто «1.».
- **Перемешать вопросы** (`--randomize`).
- **Один перенос строки перед ответом** (`--no_line_break`) — один перенос строки между вопросом и ответом вместо двух.
- **Один перенос строки после вопроса** (`--one_line_break`) — один перенос строки после текста вопроса вместо двух.
- **Игнорировать отсутствующие картинки** (`--ignore_missing_images`) — если картинка не найдена, то вместо того, чтобы падать с ошибкой, вставляет текст-заглушку.
